# 👩🏻‍💻 Lucas de Oliveira Rodrigues Souza

**`Desenvolvedora FullStack`**

Sou desenvolvedor Front-End e Full-Stack em ascensão, com foco em React.js, TypeScript, JavaScript (ES6+) e HTML5/CSS3 para construção de interfaces modernas, responsivas e escaláveis. Possuo experiência integrando APIs RESTful utilizando Express.js, com persistência de dados em MySQL e bancos relacionais, garantindo alta performance e segurança nas aplicações.

Tenho conhecimento em Node.js, NPM, Git/GitHub, componentização com React, hooks, gerenciamento de estado (Redux / Context API) e consumo de serviços externos. Também aplico boas práticas de clean code, versionamento e testes unitários para entregar soluções robustas e de fácil manutenção.

Sou entusiasta de desenvolvimento ágil (Scrum/Kanban), criação de dashboards interativos, painéis administrativos, e de projetos que combinam front-end moderno com back-end confiável, utilizando Express.js, MySQL e integração de APIs. Busco oportunidades para contribuir em projetos web desafiadores, otimizando experiência do usuário e entregando valor de forma contínua.
